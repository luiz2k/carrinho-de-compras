## 📝 Sobre
Projeto de um **Carrinho de Compras** feito no **NextJS** com **TypeScript**, consumindo a API do Mercado Livre. Na página inicial exibe todos os produtos disponíveis para adicionar ao carrinho. Após a adição de um produto, um contador aparece no canto superior direito, mostrando a quantidade de itens no carrinho. Na página do carrinho, é apresentado um resumo dos produtos adicionados, incluindo a quantidade e o preço total. Para compartilhar os dados entre a página inicial e a do carrinho, a **Context API** foi utilizada. O layout da aplicação foi construído com o auxílio do **Sass**.

## 🛠️ Tecnologias
Projeto desenvolvido utilizando as seguintes tecnologias:
- HTML / CSS / JavaScript
- TypeScript
- React.JS / Next.JS
- Context API
- Prettier / ESLint
- Sass
